# Agricultural Crop Production Analysis - Final Report

## 1. Project Overview
This project demonstrates an end-to-end crop production analytics workflow for agriculture, food security, and public policy reporting.

## 2. Business Problem
Decision-makers need to understand whether agricultural output growth is caused by harvested area expansion, yield improvement, or both.

## 3. Business Questions
- Which crops have the highest production?
- Which crops have the strongest growth?
- Which crops have high volatility?
- Which crops should be prioritized for monitoring?

## 4. Dataset
The packaged dataset is synthetic and FAOSTAT-style. It includes crop, year, production, harvested area, and yield.

## 5. KPIs
- Total production
- Harvested area
- Weighted yield
- Year-over-year growth
- Production share
- Volatility score

## 6. Latest Demo-Year Findings
- Latest year: 2024
- Total production: 8,724,019 tonnes
- Harvested area: 1,419,928 ha
- Weighted yield: 6,144 kg/ha
- Top crop: Wheat

## 7. Recommendations
1. Use yield as a key productivity KPI.
2. Monitor high-volatility crops separately.
3. Add official data before policy use.
4. Extend the project with Power BI, forecasting, and climate indicators.

## 8. Limitations
The current dataset is synthetic and cannot be used as official evidence.
