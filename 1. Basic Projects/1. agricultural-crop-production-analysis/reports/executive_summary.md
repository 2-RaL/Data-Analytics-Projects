# Executive Summary

## Project
Agricultural Crop Production Analysis

## Objective
Analyze crop production, harvested area, yield, growth, production share, and volatility using a complete portfolio-ready data analytics workflow.

## Data Note
The included dataset is synthetic and FAOSTAT-style. It is designed for training and portfolio packaging. Replace it with official FAOSTAT, Azerbaijan State Statistical Committee, World Bank, or other verified data before using it for policy decisions.

## Latest Demo Year KPIs
- Latest year: 2024
- Total production: 8,724,019 tonnes
- Total harvested area: 1,419,928 ha
- Weighted yield: 6,144 kg/ha
- Top crop by production: Wheat

## Key Insights
1. Production analysis must separate area expansion from yield improvement.
2. High-yield crops can produce large output from smaller harvested areas.
3. Volatile crops should be monitored separately for planning and risk management.
4. A dashboard should combine production, area, yield, YoY growth, and volatility.
